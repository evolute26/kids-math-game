# 👑 小小數學王 (Kids Math Master)

這是一個專為 6 歲小孩設計的互動數學練習網頁。透過鮮豔的色彩、即時的反饋以及有趣的煙火特效，讓孩子在遊戲中愛上加減法運算。

## 🌟 遊戲特點

* **分級練習**：提供「10 以內」與「20 以內」兩種難度切換，循序漸進。
* **即時反饋**：輸入正確答案後自動跳題，並觸發隨機位置的**彩色煙火特效**。
* **挑戰模式**：可自由選擇是否開啟 30 秒倒數計時，激發孩子的學習動力。
* **介面友善**：大字體、高對比色設計，適合在 iPad 或觸控螢幕上操作。

## 🛠️ 技術架構

本專案使用純前端技術開發，無需安裝任何後端環境：
* **HTML5**：網頁結構。
* **CSS3**：介面美化與動畫效果。
* **JavaScript (ES6)**：遊戲邏輯、計時器及 Canvas 煙火粒子引擎。
